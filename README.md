# TrendyWearRetailers
**Uncover your Style! Go BEYOND the usual.**

---

### Overview
**TrendyWearRetailers'** very own E-Commerce website. Created using **Next.js**, using **Supabase** as its backend service, and now with integration with **PayRex**.

This project features both an **end user** and **admin interface**.
---

### End User Features
* **Account Registration and Login** — Shoppers can create an account, log in securely, and manage their personal profile.
* **Product Browsing and Search** — Shoppers can browse the product catalog, search for items by name or category, and apply filters such as size, color, and price range.
* **Product Detail View** — Shoppers can view detailed product information including images, descriptions, available sizes, colors, stock availability, and price.
* **Shopping Cart** — Shoppers can add items to a cart, adjust quantities, remove items, and review their cart before proceeding to checkout.
* **Order Placement and Payment** — Shoppers can place orders and complete payment through the **Payrex** payment gateway, which supports Philippine-specific methods such as **GCash** and **PayMaya**.
* **Order Tracking** — Shoppers can monitor the status of their orders (e.g., Pending, Processing, Shipped, Delivered) as updated by the store manager through the admin side.
* **Wishlist** — Shoppers can save products to a wishlist for future reference or purchase.
* **Product Ratings and Reviews** — Shoppers can submit ratings and written reviews for products they have purchased.

---

### Admin User Features
* **Product Management** — Store managers can add new products, edit existing product details (name, description, price, images, stock), and remove discontinued items.
* **Inventory Management** — Store managers can update stock levels in real time and monitor inventory to prevent overselling or stockouts.
* **Order Management** — Store managers can view all incoming orders, update order statuses, and process fulfillment. Order tracking visible to the shopper is controlled entirely from the admin side.
* **Sales and Inventory Reports** — Store managers can generate automated reports on sales performance and inventory levels, supporting informed and timely business decisions.

---

### PayRex Integration
* **Webhook** — Proper Webhook integration with backend, ensuring that only **verified payments** are recognized by the system.
* **Filipino Focused payment** — Popular local modes of online payment are fully supported by PayRex.
